## 🎯 项目概述

### 系统定位
面向50-200人规模软件公司，服务四大事业部（市场/开发/项目/售后）的一体化作战指挥系统。

### 核心价值
1. **打破数据孤岛** - 统一汇总全流程数据
2. **实时进度监控** - 多维度可视化展示
3. **智能预警系统** - 自动识别并推送问题
4. **决策支撑平台** - 基于数据生成分析报表

### 目标用户
- **总指挥** - 系统最高管理者，查看全局数据
- **指导员** - 系统军师，监控所有分身智能体
- **事业部负责人** - 管理本部门数据
- **普通员工** - 查看个人任务数据

- 任务排期与跟踪

- 全局数据一览
- 一键指挥功能
- 战情分析支持
- 历史回溯查询

## 🚀 快速开始

### 环境要求
- Node.js 18+
- Python 3.11+
- Docker 24+
- PostgreSQL 15
- Redis 7

### 本地开发
```bash
# 克隆项目
git clone https://github.com/yangxy679/CIS-Command-System.git
cd CIS-Command-System

# 启动所有服务
docker-compose up -d

# 访问应用
# 前端：http://localhost:3000
# 后端API：http://localhost:3001
# 数据中台：http://localhost:8000
```

### 生产部署
```bash
# 使用部署脚本
./scripts/deploy.sh production

# 或使用Docker Compose
docker-compose -f docker-compose.prod.yml up -d
```
