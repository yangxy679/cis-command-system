## 🎯 第一版API功能

### **认证相关**
- `POST /auth/login` - 用户登录
- `POST /auth/register` - 用户注册
- `POST /auth/logout` - 用户登出
- `GET /auth/profile` - 获取用户信息

### **用户管理**
- `GET /users` - 获取用户列表
- `GET /users/:id` - 获取用户详情
- `PUT /users/:id` - 更新用户信息
- `DELETE /users/:id` - 删除用户

### **组织架构**
- `GET /organizations` - 获取组织列表
- `POST /organizations` - 创建组织
- `GET /organizations/:id/users` - 获取组织成员
